# 《巴菲特：从未亮出的那张底牌》H5游戏完整卡牌内容表

## 1. 项目概述

### 1.1 游戏机制

本H5游戏采用"投资选秀制"——系统依次向用户发出25张投资情境卡牌，用户对每张牌只能做出"投"或"过"的二选一决策，一旦选择"过"则永久跳过该机会。用户共有**5次"投"的机会**，呼应巴菲特的经典名言："人生就像一张只能打20个孔的卡片，每做一次重大投资决策就打一个孔。"

### 1.2 四类案例的游戏逻辑

|案例类型|数量|巴菲特实际行为|用户选"投"|用户选"过"|设计目的|
|:---|:---|:---|:---|:---|:---|
|成功案例|6个|投了且赚了|✅正确（获得正收益）|❌错误（错过好生意）|识别优质机会|
|失败案例|4个|投了但亏了|❌错误（承担亏损）|✅正确（避开陷阱）|识别风险信号|
|错过案例|10个|没投但错过了|✅正确（抓住机会）|❌错误（重蹈覆辙）|克服认知偏见|
|明智拒绝案例|5个|没投且庆幸没投|❌错误（承担崩盘损失）|✅正确（成功避坑）|识别危险信号|

**核心张力**：谜面采用完全中性的叙述，用户无法提前判断案例属于哪一类。最终揭晓时产生四种情绪体验：
- 选"投"命中成功案例 → 成就感（"我和巴菲特英雄所见略同！"）
- 选"投"踩中失败案例 → 警醒（"原来股神也会犯这种错"）
- 选"过"错过了错过案例 → 双重遗憾（"我竟然重蹈股神覆辙"）
- 选"过"避开了明智拒绝案例 → 智慧认同（"我也看穿了这个陷阱！"）

## 3. 设计说明与注意事项

### 3.1 卡牌发放顺序

建议在游戏中将25张卡牌打乱顺序随机发放，避免用户通过位置推测案例类型。可考虑以下平衡：
- 前5张中至少包含1张成功、1张失败、1张错过、1张明智拒绝案例
- 最后5张中保留1-2张高戏剧性案例（如#18熔断、#07伯克希尔纺织、#22雷曼兄弟）
- 明智拒绝案例（#21-#25）应分散在游戏各阶段，避免集中出现

### 3.2 图腾视觉规范

|原则|说明|
|:---|:---|
|具象化|所有图腾必须是可由插画师直接绘制的实物，禁止抽象概念|
|隐喻性|图腾与公司有内在关联，但不能是公司logo或直接产品|
|统一风格|建议采用"复古金融插画"风格——深绿色背景、金色描边、做旧质感|
|尺寸适配|图腾需适配卡牌正面（约200×280px）和缩略图（64×64px）两种尺寸|

### 3.3 谜面朗读时长控制

每张谜面设计在150-200字，以正常语速朗读约30-40秒。如需配合语音导览或动画演示，建议：
- 第一段（背景铺垫）：15-20秒
- 第二段（核心矛盾）：10-15秒
- 第三段（选择题）：5秒

### 3.4 收益率计算规则

建议的计分机制：

|案例类型|用户选"投"|用户选"过"|
|:---|:---|:---|
|成功案例|获得该案例实际回报率（如可口可乐+2000%）|0收益|
|失败案例|获得该案例实际亏损率（如康菲石油-37%）|0收益|
|错过案例|获得假设投资的潜在回报率（如谷歌+10000%）|0收益|
|明智拒绝案例|获得该案例实际崩盘损失率（如雷曼-100%）|+10%（避坑奖励）|

最终段位根据5次"投"的累计收益率计算，参见主策划方案中的段位系统。
